# Configurable slayer info

A RuneLite plugin that lets you configure information to be shown when given slayer tasks.

# Features

- Overlay box with information configurable by the user. Newlines created bullet points in the overlay
- Automatically hides the information box when you are in the area of your current slayer task
- Debugging to see how the overlay will look like

# Screenshots

Examples of how default values are specified for Ankous and Cockatrices and how it looks in the overlay. You can freely change the values for any monsters and test it using the "Debugging".
